#Project Description

This project implements a distributed system using gRPC. It provides three services:

-Electricity Usage Service
-Grocery Management Service
-Water Consumption Service

A naming service is used for service discovery.

#Technologies Used

Java  
gRPC  
Maven  
JmDNS  
Netbeans

#Project Structure

src/main/java → service implementations  
src/main/proto → proto definitions
